# Notes App UI – React Native

A clean and responsive Notes App UI built using React Native and Expo.

## Features

- Notes listing screen
- Note editor screen
- Dark/Light mode support
- Search functionality
- Add new notes
- Responsive layout
- Keyboard avoiding support
- Floating action button

## Components Used

- FlatList
- TextInput
- Pressable
- Switch
- KeyboardAvoidingView
- ImageBackground
- SafeAreaView

## Hooks Used

- useState
- useColorScheme
- useWindowDimensions

## Styling

- StyleSheet.create()
- StyleSheet.compose()

## Additional Improvements

- Functional add-note flow
- Shared theme management
- Responsive UI design
- Floating add button
- Safe area handling
- Real-time note updates
